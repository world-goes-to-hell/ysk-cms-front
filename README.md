# YSK CMS Frontend

YSK CMS 프론트엔드 프로젝트

## 기술 스택

- Vue 3.5
- Vite 7
- TypeScript 5.9
- Vue Router 4
- Pinia 3
- Element Plus 2
- Axios

## 프로젝트 설정

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

### 프로덕션 빌드

```bash
npm run build
```

### 타입 체크

```bash
npm run type-check
```

### Lint

```bash
npm run lint
```
